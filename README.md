# Cinema Game — Frontend

Next.js frontend for [Cinema Game](https://github.com/ChiPowers/cinema_game), a movie trivia game where you connect actors through shared movies.

## Stack

- Next.js
- Tailwind CSS
- Framer Motion
- Cloudflare Pages (hosting)

## Getting started

```bash
npm install
npm run dev
```

UI runs at `http://localhost:3000` (or next available port).

Set `NEXT_PUBLIC_API_URL` in `.env.local` to point to the backend API (defaults to `http://localhost:8000`).
